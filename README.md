# Where We Go

## Prerequisites

- .NET 10.0 SDK++

## Command for Project

### 1. Restore Dependencies
```bash
dotnet restore
```

### 2. Build Project 
```bash
dotnet build
```

### 3. Run Project
```bash
dotnet run
```

### 4. Run in Watch Mode (Auto-reload)
```bash
dotnet watch run
```

### 5. Clean Build Files 
```bash
dotnet clean
```
